## 運用ルール

- ステータスは `Backlog / Planned / In Progress / Review / Done / Blocked` のいずれかを使う。
- `Done` はコード、DB migration、RLS、主要な正常系・異常系、本番確認が完了した状態とする。
- AIは採用判断を代替せず、候補者の自動不採用を行わない。
- 公開情報、利用規約、robots.txt、保存権、取得頻度をデータソースごとに確認する。
